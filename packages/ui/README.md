# UI

The graph displayed using [vis.js](https://visjs.org/).

## Features

* Graph view:
  * Nodes and relationships
  * Open node view by clicking on it
  * Filter nodes by:
    * Tags
    * Name
* Node view:
  * Name
  * Tags
  * Link to git repository
  * Dependencies:
    * Relationship
    * Related node
    * Link to source code
